# Cosmic Blast — Boss Rush 弹幕射击游戏

Godot 4 弹幕射击 Boss Rush 游戏。玩家操控战机与四大 Boss 家族战斗。

## Boss 家族

### 1. 天堂号 (Paradise) — 浮空要塞

大型浮空战舰，停泊在屏幕顶部/左/右边缘，携带 4 门独立炮台。

| 亚种 | 技能池 | 冷却 |
|------|--------|------|
| 天堂号 (Paradise) | 1, 2, 3, 4, 5, 6 | 1.5s |
| 乌托邦 (Utopia) | 1, 3, 4, 5 | 2s |
| 伊甸园 (Eden) | 1, 3, 4, 5, 6 | 1s |
| 桃源乡 (PeachBlossom) | 1, 2, 3 | 3s |

**核心机制：**
- 4 门独立炮台，各有独立 HP（MaxHP/4）、独立技能冷却
- 炮台摧毁后爆炸压暗、不再旋转、不再发射激光、不再接受伤害
- 存活炮台可发动技能并扣除 Boss HP
- LEFT/RIGHT 停泊时摇曳幅度限制，确保船体不越屏
- 全场均匀伤害（所有炮台受击 → Boss HP 扣减）

**技能概览：**
| # | 名称 | 描述 |
|---|------|------|
| 1 | 集中射击 | 所有炮台对玩家齐射 |
| 2 | 旋转扫射 | 炮台旋转扫描扇形射击 |
| 3 | 飞掠轰炸 | 飞船飞过屏幕中央轰炸 + 召唤敌机 |
| 4 | 扫射风暴 | 炮台持续旋转扫射 6s |
| 5 | 散弹扫射 | 6-10 发扇形散弹覆盖 60-90° |
| 6 | 能量投射 | 炮台飞至中央蓄力 → 十字爆炸 → 回到原位 |

### 2. 扭曲星核 (WarpedCore) — 脉冲能量球

带 4 个环绕小球的脉动能球，布朗运动待机，展开/收缩/冲刺。

| 亚种 | 技能池 | 冷却 |
|------|--------|------|
| 扭曲星核 (WarpedCore) | 1, 2, 3, 4, 5, 6 | 2s |
| 异变源石 (Source) | 1, 2, 3 | 3s |
| 诡异菌孢 (Spore) | 1, 2, 3, 4 | 2s |
| 反物质核 (Anti) | 2, 3, 4, 5, 6 | 1s |

**核心机制：**
- 1.5x 基础缩放（主体/碰撞/环绕球半径）
- 四球各自独立半径峰值（技能2 中 3-6× 随机）
- 技能执行期间受到伤害减半
- AI：敌机 >8 禁止技能3；敌机 ≤3 强制技能3

**技能概览：**
| # | 名称 | 描述 |
|---|------|------|
| 1 | 停泊位转移 | 3s 警戒 → 加速冲刺到新停泊位（1s 蓄力） |
| 2 | 轨道扩张 | 咆哮膨胀 → 轨道扩张 5s + 维持 3s + 收缩 5s，全程震动 |
| 3 | 咆哮召唤 | 0.25s 膨胀 → 全屏震动 3s + 召唤 5-10 敌机 |
| 4 | 强吸力咆哮 | 膨胀 + 10s 吸力动画（粒子 + 残骸卷入） |
| 5 | 十字激光连射 | 20-30 枚能量球依次发射至全屏随机位置 → 蓄力 → 十字激光 |
| 6 | 连续冲刺 | 8-12 次向玩家冲刺（警戒时间递减 2s→1s）→ 末次重选停泊位 |

### 3. 地狱之眼 (HellEye) — 凝视之眼

独立 Boss（非扭曲星核子类），红色星云 + 眼珠 shader 裁剪，眨眼/瞪眼/眯眼动画。

| 亚种 | 技能池 | 冷却 |
|------|--------|------|
| 地狱之眼 (HellEye) | 1, 2, 3, 4, 5, 6 | 2s |
| 前哨战 (Sentry) | 1, 2, 3 | 3s |
| 防火墙 (Admin) | 2, 3, 4, 5 | 2s |
| 典狱长 (Gate) | 2, 3, 4, 5, 6 | 1s |

**核心机制：**
- 150% 全局缩放（mask/nebula/eyeball/碰撞半径）
- 闭眼 → 无敌（技能3）；睁眼 → 受击可扣血
- 技能4 死时自动还原 `GameManager.controls_inverted`
- 技能2 圆环间隔 1-2s，修复假命中分支
- MiniHellEye 出生带睁眼动画

**技能概览：**
| # | 名称 | 描述 |
|---|------|------|
| 1 | 眯眼激光 | 5-10s 眯眼，以玩家为中心随机角度 2 条十字激光 |
| 2 | 瞪眼收缩环 | 围绕玩家收缩 3-6 个缺口圆环（60-90°缺口），触碰扣血 |
| 3 | 闭眼传送 | 闭眼 3-5s → 随机传送 → 睁眼，三面生成 8-12 敌机 |
| 4 | 红幕扭曲 | 瞪眼 → 扩散圆环 → 全屏红幕 8-12s + ERROR 文字 + 方向反转 |
| 5 | 睁眼连射 | 同技能1，每次 3 条激光，间隔 0.27-0.53s |
| 6 | 瞪眼分身 | 5-8s 生成 8-12 个 MiniHellEye（HP=2%，睁眼动画后持续吸玩家 HP） |

## 技术栈

- **引擎**: Godot 4 (GDScript)
- **分辨率**: 1920×1080
- **模块化**: Boss 基类 → 亚种重写 `_ready()` 覆盖导出变量，所有核心逻辑通过继承复用

## 文件结构

```
scenes/        — 游戏场景和 Boss 预制体 (.tscn)
scripts/       — GDScript 脚本
assets/
  images/      — Boss 贴图、特效、UI
  audio/       — BGM、音效
  helleye/     — 地狱之眼专有资源 (mask/shader/variants)
```
